## **🐤 Canary.io**

Uma prova-de-conceito que visa implementar mais segurança e confiabilidade na forma como imagens são expostas na web.

---

### **🚀 Como executar o projeto:**
Basta abrir o terminal e executar o seguinte comando:


```bash
$ docker compose up
```
